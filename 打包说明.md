# Story KB 单机包说明

## 启动

双击：

```text
start-story-kb.bat
```

程序会启动本地数据库服务，并自动打开：

```text
http://127.0.0.1:5178/
```

## 数据保存位置

知识库主数据库：

```text
database/kb.json
```

统计与清单：

```text
database/manifest.json
```

浏览器页面会通过本地接口写入 `database/kb.json`。只要打包时保留整个目录，知识库数据会跟随程序一起移动。

## 目录含义

```text
assets/       资源目录
database/     本机知识库数据
runtime/      本地运行服务和依赖
src/          客户端界面与知识拆解逻辑
index.html    客户端入口页面
```

## 不需要打包的内容

以下内容已经从便携包中排除：

```text
runtime/edge-test-profile/
wireframes/
tests/
早期调研原始文件
```
